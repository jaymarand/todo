# Todo App (iOS)

A production-ready iOS Todo app built with SwiftUI and SQLite (C API).

## Features
- Complete CRUD for Todo items.
- Filter by All / Active / Completed.
- Swipe actions: Delete, Complete/Mark Active.
- Local persistence with SQLite (no 3rd party libs).
- Sorting: Active first, then createdAt desc; Completed sorted by completedAt desc.

## Requirements
- iOS 16.0+
- Xcode 15.2+

## Setup & Workflow

### On Windows (Development)
Generate and edit code on Windows.

To push changes:
```powershell
git add .
git commit -m "Initial Todo app"
git push
```

### On Mac (Build & Run)
SSH into your Mac mini and pull the latest code.

```bash
cd ~/projects/todo
git pull
open Todo.xcodeproj
# Or if generated by Xcode via command line
# xcodebuild -scheme Todo -destination 'platform=iOS Simulator,name=iPhone 15 Pro' build
```

Then in Xcode:
1. Product > Run (Simulator)
2. Product > Archive -> Distribute -> TestFlight
